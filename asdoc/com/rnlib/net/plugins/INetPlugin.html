Interfacepublic interface INetPlugin extends flash.events.IEventDispatcher
Implementors FileReferencePlugin

Developer of Plugin is responsible for dispatch NetPluginEvent.READY after executed by RemoteAmfService method init(). The event inform amf service that plugin is ready for return args which will pass to request on server by args getter.
