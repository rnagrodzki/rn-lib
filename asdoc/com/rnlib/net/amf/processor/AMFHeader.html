Classpublic class AMFHeader

Implements IDisposable

An AMF packet level header. Request and response headers have an identical structure. When making use of by-reference serialization, the reference tables are reset for each AMFHeader. Note that the header name String is never sent by reference and does not participate in by-reference serialization.
