mock () function  
public function mock(target:*):MockingCouverture

Create an expectation of required behaviour on a target instance. When the target instance was created using nice() and a method or property on the target instance is called and no expectations have been set using mock() or stub() then a false-y value will be returned (false, null, 0, NaN) When the target instance was created using strict() and a method or property on the target instance is called and no expectations have been set using mock() or stub() then an UnspecifiedBehaiourError will be thrown. When verify() is called for the target instance and the expectation has not been met then an UnmetExpectationError will be thrown.

Parameters

target:* — Object created by nice() or strict()

Returns
MockingCouverture

See also

mockolate.stub()
mockolate.nice()
mockolate.strict()

Example 
     var flavour1:Flavour = make(Flavour);
     var flavour2:Flavour = make(Flavour);
     var combined:Flavour = make(Flavour);
     
     mock(flavour1).method('combine').args(flavour2).returns(combined);
     
     var result:Flavour = flavour.combine(flavour2);
