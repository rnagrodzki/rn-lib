Classpublic class MockolateRule
InheritanceMockolateRule org.flexunit.internals.runners.statements.MethodRuleBase
Implements org.flexunit.rules.IMethodRule

MockolateRule is the recommended way to enable Mockolate support in testcases using FlexUnit 4.1 and up. Use [Mock] metadata to mark public vars that should be injected with a Mockolate instance.
